│           │ lw       │ Load 32-bit value (int)            │

│           │ lb       │ Load 8-bit signed                  │
├───────────┼──────────┼────────────────────────────────────┤

One-line mental rules (remember these)
lw / ld   → may page-fault
ecall     → user → kernel
sret      → kernel → user
a7        → syscall number
a0        → return value
zero (x0) → always 0 (accessing 0(zero) == NULL)
