# XV6
The xv6 operating system is a simple, Unix-like teaching operating system developed at MIT. It is designed to run on RISC-V architecture and is often used in academic settings to teach operating system concepts.
ref: [xv6-public](http://pdos.csail.mit.edu/6.828/2021/xv6.html)

```
┌───────────────┐
│      ROM      │  (M-mode, SoC code)
└───────┬───────┘
        │ loads
        ▼
┌───────────────┐
│    OpenSBI    │  (M-mode runtime)
└───────┬───────┘
        │ jumps to
        ▼
┌────────────────┐
│     U-Boot     │ (S-mode bootloader)
│   Loads xv6    │
└───────┬────────┘
        │ booti / bootm
        ▼
┌────────────────┐
│      xv6       │ (S-mode OS)
│    Userland    │
└────────────────┘
```
